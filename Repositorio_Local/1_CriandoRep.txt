Dono do reposiótio:
git init
-> Iniciar um repositório Git em seu projeto.

git add <arquivos>
-> Colocar arquivos na área de staging

git commit -m "<mensagem>"
-> Colocar arquivos/alterações na área de staging

git branch
-> Verifica as branch
git branch <nome>
-> Cria uma branch com nome <nome>

git checkout <nome_branch>
-> Altera para a branch com nome branch_nome

git remote add origin https://github.com/MarcosPaulo10/Teste_Apresentacao.git
-> Conecta o repositório local ao repositório remoto
